# Mini App Expedition

Bienvenue dans le projet **Mini App Expedition**, un mini-projet collaboratif conçu par l’équipe **Nirida Tech** pour apprendre à développer des applications (ici version desktop) avec Flutter et le langage Dart.

---

##  Objectif du projet

Ce projet a pour objectif de permettre aux membres de l’équipe de :

- Découvrir la structure d’un projet Flutter
- Comprendre les bases du langage Dart
- Créer des interfaces utilisateur modernes
- Apprendre la navigation entre plusieurs pages
- Manipuler des données locales
- Collaborer efficacement avec Git et GitHub
- Suivre un workflow professionnel basé sur les branches

---

##  Description de l’application

L’application **Xpedition** est une mini application desktop composée de **4 pages principales**.

L’objectif n’est pas de construire une application complète, mais de se familiariser avec Flutter et Dart à travers un projet pratique et collaboratif.

### Pages à réaliser

1. Dashboard
2. Suivi colis
3. Calculatrice
4. Contacts

---

##  Technologies utilisées

- Flutter
- Dart
- Git
- GitHub
- Visual Studio Code ou Android Studio

---

##  Organisation de l’équipe

Chaque développeur travaille sur une branche dédiée :

- `feature/ui-dashboard`
- `feature/suivi-page`
- `feature/calculatrice-page`
- `feature/contact-page`

Toutes les fonctionnalités sont fusionnées dans la branche `develop`, puis validées avant d’être intégrées dans `main`.
